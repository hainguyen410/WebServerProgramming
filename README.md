A Dynamic Website for Renter and Borrower to borrow and manage rental car. This is my projects in Web Server Programming Subject at UOW.

 <img width="1680" alt="image" src="https://github.com/iminloved/WebServerProgramming/assets/75488384/6f2ab7e1-6263-435d-9c2d-3100cf3d2828">
<br/><br/>

For users not already have an account yet. You can press the register link to create a new account.

<img width="1680" alt="image" src="https://github.com/iminloved/WebServerProgramming/assets/75488384/c68f4ba5-a7f4-4454-bb9c-e43fa65a3bb2">
<br/><br/>

In the registration link, you can choose to be an administrator or a user. Once selected, you will be able to access the website corresponding to the user type.
<br/>

For the admin page: There are 3 actions for an admin to execute: Insert a Car, Search for a car, and Change status of the car. Each time you press the button, the form of that action will display on the page accordingly.
